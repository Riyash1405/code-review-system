# Code Review System

A scalable code analysis platform integrated with GitHub that evaluates code quality, detects issues, and provides actionable insights.

## 🚀 Features
- GitHub OAuth integration
- Repository analysis
- Code quality scoring
- Async job processing

## 🛠 Tech Stack
- Backend: Node.js, Express
- Frontend: React ,TypeScript,TailwindCSS
- Database: PostgreSQL
- Queue: Redis
